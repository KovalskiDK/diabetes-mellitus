# Релизы GI Index

Здесь хранятся APK файлы приложения для ручной загрузки.

## Последние версии:

- **v1.9.6** - Автоматическая проверка обновлений
- **v1.9.5** - Защита от прерывания загрузки
- **v1.9.4** - Исправлено автообновление
- **v1.9.3** - Автозагрузка данных при установке

## Установка:

1. Скачайте APK файл из раздела [Releases](https://github.com/KovalskiDK/diabetes-mellitus/releases)
2. Разрешите установку из неизвестных источников в настройках Android
3. Откройте скачанный APK и установите приложение

## Автообновление:

Приложение поддерживает автоматическое обновление без Google Play:
- Автоматическая проверка обновлений каждый час
- Ручная проверка в настройках приложения
- Загрузка и установка одним нажатием
