# GI Database Parser

Парсер для сбора данных о гликемических индексах с glycemicindex.com

## Установка

```bash
pip install -r requirements.txt
```

## Использование

```bash
python gi_scraper.py
```

Скрипт:
1. Проверит первую страницу
2. Покажет примеры найденных продуктов
3. Спросит подтверждение для полного парсинга
4. Соберет все продукты со всех страниц
5. Сохранит в `gi-database-raw.json`

## Следующие шаги

После парсинга нужно:
1. Запустить `translate_products.py` для перевода названий на русский
2. Проверить и очистить данные
3. Конвертировать в финальный формат для приложения
